# SeatingChart

Randomly generate a seating chart.

# DEMO

![demo](https://github.com/ytyaru/Shell.SeatingChart.20200925092353/blob/master/demo/demo.png?raw=true)

# Features

* You can specify the number of people
* You can specify the seating order
    * Attendance number order
    * Random
    * Random (alternate between men and women)
